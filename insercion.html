Método de Ordenamiento por Inserción

Objetivo

Brindar al visitante información sobre el método de ordenación por inserción, su algoritmo de funcionamiento y aplicar su algoritmo de implementación en algunos lenguajes de programación.

Desarrollo

Es uno de los métodos de ordenación que hacemos cuando jugamos cartas, cuando tenemos la mano de cartas, vamos organizándolas tomando las castas de izquierda a derecha, una a una vamos comparándola con la anterior y si es menor la movemos a la posición y corremos la siguiente carta. Ejemplo:

Tenemos las siguientes cartas. Vamos a ordenarlas de menor a mayor, comenzando con el 7 de la izquierda, lo dejamos en su lugar y continuamos con el 6 de la posición 1, comparamos con el anterior 7, es menor entonces lo movemos a la posición 0 y el 7 lo corremos a la derecha,

Seguimos con la siguiente el 4 en la posición 2, comparamos con la anterior el 7, es menor, seguimos comparándola con la siguiente a la izquierda, es la carta del 6, es menor, sacamos el cuarto de la posición 2 y corremos el 7 y el 6 a la derecha, ubicamos el 4 en la posición 0.

Seguimos con la siguiente carta el 2 en la posición 3, comparamos con el anterior la carta 7, como es menor, seguimos comparando con el anterior el 6 como es menor, seguimos comparando con el anterior el 4 como es menor y no hay más comparaciones, sacamos el 2 de la posición 3 y corremos a la derecha las cartas anteriores e insertamos la carta 2 en la posición 0.

Ahora seguimos con la carta 9, comparamos con la anterior, como es mayor la dejamos en la posición y seguimos con la siguiente, la carta 8, comparamos con la anterior la carta 9 como es menor, comparamos con la siguiente la carta 7, como es mayor ya tenemos la posición, sacamos la carta 8 de la posición 5, corremos la carta 9 de la posición 4 a la posición 5, e insertamos la carta 8 en la posición 4.

Seguimos con la carta 3 en la posición 6, sacamos la carta de la posición y comparamos con la anterior la carta 9, como es menor, comparamos con la siguiente la carta 8, como es menor seguimos con la carta 7, como es menor seguimos con la carta 6, como es menor seguimos con la carta 4, como es menor seguimos con la carta 2, como es mayor encontramos la posición 1, corremos las demás cartas hacia la derecha e insertamos la carta 3 en la posición 1.

Continuamos con la carta 5, sacamos la carta de la posición 7 y comparamos con la anterior la carta 9, como es menor, comparamos con la siguiente la carta 8, como es menor, comparamos con la carta 7, como es menor comparamos con la carta 6, como es menor comparamos con la carta 4, como es mayor determinamos la posición es 3, corremos las demás cartas hacia la derecha en sus posiciones e insertamos la carta 5, ya que no hay más cartas a ordenar, tenemos la mano de cartas ordenadas.

Algoritmos

  • C# (C Sharp) 
using System;
using System.Collections.Generic;
using System.Linq;
using System.Text;

namespace PruebaVector
{
    class PruebaVector
    {
        private int[] vector;

        public void Cargar()
        {
            Console.WriteLine("Metodo de insercion directa");
            Console.Write("Cuantos longitud del vector:");
            string linea;
            linea = Console.ReadLine();
            int cant;
            cant = int.Parse(linea);
            vector = new int[cant];
            for (int f = 0; f < vector.Length; f++)
            {
                Console.Write("Ingrese elemento "+(f+1)+": ");
                linea = Console.ReadLine();
                vector[f] = int.Parse(linea);
            }
        }

        public void InsercionDirecta()
        {
            int auxili;
            int j;
            for (int i = 0; i < vector.Length; i++)
            {
                auxili = vector[i];
                j = i - 1;
                while (j >= 0 && vector[j] > auxili)
                {
                    vector[j + 1] = vector[j];
                    j--;
                }
                vector[j + 1] = auxili;
            }
        }

        public void Imprimir()
        {
            Console.WriteLine("Vector ordenados en forma ascendente");
            for (int f = 0; f < vector.Length; f++)
            {
                Console.Write(vector[f]+"  ");
            }
            Console.ReadKey();
        }


        static void Main(string[] args)
        {
            PruebaVector pv = new PruebaVector();
            pv.Cargar();
            pv.InsercionDirecta();
            pv.Imprimir();
        }
    }
}
